
# WORLD TIMES

Ülke Ve Şehirleri filtreleyerek dünya saatlerini gösteren uygulama

## Features

- Light/dark mode toggle
- Searchable
- Splash Screen
- Cross platform


## API Reference

```http
  GET  http://worldtimeapi.org
```


## Tech Stack

**Client:** Flutter 

**Technologies:**

- Bloc/Cubit
- Provider
- MVVM
- REST API
- Dio
